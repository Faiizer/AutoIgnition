# AutoIgnite

With this plugin, furnaces and campfires will start automatically whenever both fuel and input items are placed inside, so you no longer need to press the 'Turn on' button manually.

## Installation

1. Navigate to Hytale Mods folder:  
   `C:\Users\YourUsername\AppData\Roaming\Hytale\UserData\Mods\`
2. Copy the plugin JAR into the `Mods/` folder.
3. Launch Hytale.
4. Enable the mod in the world settings.
